## New Workflow (Much Faster)

```

2. Click "Translate + Preview" button
3. Wait for result
4. Edit translation
5. Click "✓ Save Card"

### New Way (Keyboard-based)
```

## If Translation Still Looks Wrong

1. **Edit manually** (you can!)
   - Translation appears in English field
   - Edit it
   - Click Save
   - Your version is saved!

2. **Try rephrasing**
   - "überzeugt" works better than "überzeugter"
   - Singular better than plural
   - Base form better than conjugated

3. **Examples still help**
   - Even if AI translation is weird
   - Examples show real usage
   - Context helps you learn

## Customization Options

### Want Different Accent Color?
Edit `styles.css`, find:
```css
--primary: #6366f1;
```

Change to:
- Blue: `#3b82f6`
- Green: `#10b981`
- Purple: `#8b5cf6`

### Want Darker Background?
Edit:
```css
--bg: #f5f7fa;
```

Change to:
- Slightly darker: `#f0f1f5`
- More gray: `#e8e9f0`

## Troubleshooting

### Theme Not Changing?
- **Fix**: Hard refresh (Ctrl+Shift+R or Cmd+Shift+R)
- Clear browser cache

### Ctrl+Enter Not Working?
- **Check**: Are you in the textarea?
- **Try**: Make sure textarea has focus
- **Note**: Ctrl on Windows/Linux, Cmd on Mac

### Translation Still Bad?
- **Option 1**: Edit manually (allowed!)
- **Option 2**: Try different word
- **Option 3**: Use Examples to understand meaning

### Theme Too Light?
- **Option 1**: Adjust your monitor brightness
- **Option 2**: Enable OS dark mode (light theme still works)
- **Option 3**: Edit colors in styles.css
